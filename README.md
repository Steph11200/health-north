Projet  – Health North
BTS SIO – Option SLAM

Auteur : [Stéphanie DEVILLE]
Année : 2025-2026

-------------------------------------
Description du projet
-------------------------------------

Le projet Health North est une application permettant la gestion de rendez-vous médicaux.

Le système est composé de plusieurs éléments :

1. Une API REST développée en Node.js
2. Une base de données MariaDB
3. Une application web permettant la prise de rendez-vous
4. Une application lourde (Windows Forms C#) utilisée par les spécialistes
5. Une application mobile développée en React Native permettant aux patients de consulter leurs informations.

-------------------------------------
Technologies utilisées
-------------------------------------

Backend :
- Node.js
- Express
- JWT
- MariaDB

Infrastructure :
- Docker
- OpenLDAP

Frontend :
- HTML / CSS / JavaScript
- React Native (Expo)

Application lourde :
- C#
- Windows Forms
- Visual Studio

-------------------------------------
Fonctionnalités principales
-------------------------------------

- Authentification des utilisateurs
- Gestion des patients
- Gestion des rendez-vous
- Consultation des prescriptions
- Génération de factures PDF
- Consultation des rendez-vous via application mobile

-------------------------------------
Instructions d'installation
-------------------------------------

1. Lancer Docker pour démarrer la base de données et LDAP
2. Lancer l'API Node.js
3. Ouvrir l'application web avec live server Visual Studio Code
4. Ouvrir l'application mobile avec Expo
5. Ouvrir l'application lourde avec Visual Studio
